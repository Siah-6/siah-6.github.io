What are Embeddable Widgets?

Embeddable widgets are self-contained web components that can be integrated into any website using iframes. They're perfect for sharing functionality across multiple sites while maintaining consistent branding and behavior.

Simple iframe embedding with just a few lines of code

Responsive Design

Works seamlessly across all devices and screen sizes

Secure & Isolated

Iframe sandboxing provides security and prevents conflicts

How to Embed This Widget

Basic iframe Embedding

The simplest way to embed the widget:

<iframe 
    src="https://yoursite.com/widget/index.html" 
    width="400" 
    height="500"
    frameborder="0"
    scrolling="no"
    title="Weather Widget">
</iframe>

Responsive Embedding

Make your widget responsive with CSS:

<div class="widget-container">
    <iframe 
        src="https://yoursite.com/widget/index.html" 
        width="100%" 
        height="500"
        frameborder="0"
        scrolling="no"
        title="Weather Widget">
    </iframe>
</div>

<style>
.widget-container {
    max-width: 400px;
    margin: 0 auto;
    border-radius: 10px;
    overflow: hidden;
    box-shadow: 0 4px 20px rgba(0,0,0,0.1);
}

.widget-container iframe {
    display: block;
    border: none;
}
</style>

Advanced Integration

Dynamic loading with JavaScript:

function loadWeatherWidget(containerId, options = {}) {
    const container = document.getElementById(containerId);
    const iframe = document.createElement('iframe');
    
    iframe.src = 'https://yoursite.com/widget/index.html';
    iframe.width = options.width || '400';
    iframe.height = options.height || '500';
    iframe.frameBorder = '0';
    iframe.scrolling = 'no';
    iframe.title = 'Weather Widget';
    
    // Add loading spinner
    container.innerHTML = '<div class="loading">Loading widget...</div>';
    
    iframe.onload = () => {
        container.innerHTML = '';
        container.appendChild(iframe);
    };
    
    iframe.onerror = () => {
        container.innerHTML = '<div class="error">Failed to load widget</div>';
    };
}

// Usage
loadWeatherWidget('widget-container', {
    width: '100%',
    height: '500'
});
